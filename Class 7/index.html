Debouncing

Debouncing ensures that a function is only executed after a certain amount of time has passed since it was last called. This is useful for search inputs, window resizing, or auto-saving.

function debounce(func, wait) { let timeoutId; return function(...args) { clearTimeout(timeoutId); timeoutId = setTimeout(() => { func.apply(this, args); }, wait); }; }

Throttling

Throttling ensures a function is called at most once in a specified time interval, no matter how many times the event is triggered. Useful for scroll events, mouse movements, or window resizing.

function throttle(func, limit) { let lastCall = 0; return function(...args) { const now = Date.now(); if (now - lastCall >= limit) { lastCall = now; return func.apply(this, args); } }; }

Key Differences

Debounce groups multiple sequential calls into a single one. It's like waiting for someone to finish speaking before responding.

Throttle ensures a function runs at most once per specified time interval. It's like a rate limiter that ensures a function doesn't run too often.
